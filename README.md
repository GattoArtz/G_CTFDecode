# Decode-for-CTF
CTF用スコアリング付きのHex , Base64 , URL Encoding変換ツール

## できること
・入力された文字列をHex,Base64,URL Encodingとして受け取りデコード
・デコード結果をスコアリングし、上位スコアのデコード結果を一つピックアップして表示
※フラグフォーマットを入力することでフラグフォーマットに沿ったデコード結果にスコアが加算されます


## 仕組み
・単純にHex,Base64,URL Encodingとしてデコード<br>
↓<br>
・デコード結果に可読でないと判断される文字列(現状では\のみ)が含まれた場合、その割合の分スコアを減らす<br>
↓<br>
・フラグフォーマットが入力されていた場合は、そのフラグフォーマットに沿ったデコード結果にスコアを加算<br>
・フラグフォーマットが入力されていない場合は、デコード結果に"{"と"}"が含まれていたかに応じてスコアを加算<br>
↓<br>
・デコード結果とRecommendを表示<br>

## 動機
・CTFをやっていると毎度毎度CyberChefに持っていくのが面倒だ！じゃぁ自分で作ってしまおう！の気持ちで実装。<br>
